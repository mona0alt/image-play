惩罚机制

  • 用户每次审核不通过（原图或结果图），计数 +1
  • 累计达到 3 次，触发 10 分钟封禁
  • 封禁期间：禁止上传原图、禁止提交任何生成请求
  • 封禁记录写入数据库，10 分钟后自动解封
  • 解封后计数清零

数据库补充：audit_bans（审核封禁记录表）

id              BIGINT PK AUTOINCREMENT
user_id         BIGINT NOT NULL REFERENCES users(id)
reason          VARCHAR(255) NOT NULL           -- 'audit_rejected_3_times'
banned_at       TIMESTAMP DEFAULT NOW()
expires_at      TIMESTAMP NOT NULL              -- banned_at + 10 minutes
is_active       BOOLEAN DEFAULT TRUE
created_at      TIMESTAMP DEFAULT NOW()

说明： 查询时检查 user_id 是否有 is_active = true 且 expires_at > now() 的记录。
