小程序页面结构 V2（参考 ChatGPT 风格）

极简主界面 + 抽屉/弹窗收纳次要功能

设计原则

  • 一个主界面：所有生成功能集中在首页，不用 Tab 切换
  • 历史藏侧边栏：像 ChatGPT 一样，历史记录收在左侧抽屉
  • 个人入口收右上角：余额、充值、设置收在头像下拉菜单
  • 结果全屏展示：生成结果以底部滑出卡片或全屏页展示
  • 底部悬浮操作：核心按钮（生成/重新生成）始终在手边

页面清单

1

首页（主生成界面）

顶部栏（极简）：

  • 左上角 ☰ 按钮 → 打开历史抽屉
  • 中央标题：可切换 "文生图" / "图生图" 的小下拉
  • 右上角头像 → 打开个人菜单

中央区域：

  • 大输入框（占位符："描述你想要的画面..."）
  • 图生图模式下，输入框上方显示原图缩略图（可点击重选）
  • 风格选择：输入框下方一排小圆角标签（写实 / 动漫 / 油画 / 像素...）

底部悬浮：

  • 圆形/胶囊生成按钮，始终可见
  • 按钮旁小字显示预估费用（"本次 1 元" / "免费额度剩余 2 张"）
2

历史抽屉（侧边栏）

从左侧滑出，遮罩层覆盖主界面。

  • 顶部：搜索框 + "新建生成"按钮
  • 中部：按时间分组（今天 / 昨天 / 7天内 / 更早）
  • 每项：缩略图 + prompt 前 10 字 + 时间
  • 长按/左滑：删除（软删除）
  • 点击：关闭抽屉，回到首页并加载该 prompt 到输入框
3

个人菜单（弹窗）

点击右上角头像后下拉/弹出。

  • 头像 + 昵称
  • 余额：¥12.50
  • 免费额度：剩余 1 张
  • ---
  • 去充值 → 跳转充值页
  • 生成历史 → 打开历史抽屉
  • 隐私政策
  • 用户协议
4

结果预览页（全屏/底部卡片）

生成完成后从底部滑出全屏卡片，或跳转新页面。

  • 大图展示生成结果（1024x1024）
  • 底部操作栏：
    • 💾 保存到相册
    • 📤 分享给好友
    • 🔄 重新生成（用相同 prompt）
    • ✏️ 修改 prompt（返回首页并保留输入）
  • 下滑/点击遮罩关闭，回到首页
5

充值页（全屏）

从个人菜单进入。

  • 顶部：当前余额
  • 中部：套餐卡片（充50送10 / 充100送30）
  • 底部：微信支付按钮
  • 支付成功后自动返回并刷新余额

关键交互细节

  • 首次进入：直接展示首页，无需选择。默认文生图模式。
  • 文生图 ↔ 图生图切换：点击顶部标题栏的下拉箭头切换，不用跳转页面。
  • 生成中：输入框禁用，按钮变加载态，显示进度文案（"正在想象画面..."）。
  • 连续生成：结果页下滑关闭后，输入框保留上次 prompt，方便修改后再生成。
  • 空状态：历史抽屉为空时，显示"还没有生成记录，去创作第一张吧"。
